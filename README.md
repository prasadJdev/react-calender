# react-calender
Calender app to keep track of Todos, Events, Tasks

# Description:
- User can add a new event, a todo or a task

# Dependencies:
- fullcalendar 

# Instalation:
- Project created useing Vite
- npm i to install required pakages
- npm run dev to start development server
